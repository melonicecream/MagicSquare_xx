# MagicSquare_xx — Mom Test 인터뷰 질문 10개

**페르소나:** 4×4 격자, 빈칸 2개(0), 1~16, 행·열·대각선 합 34 맞추는 학습자  
**원칙:** 과거 사실·구체적 행동만 묻기 / 솔루션·기능·미래 의견 금지

---

## 1. 마지막으로 틀렸을 때

### Q1
**「4×4 마방진에서 마지막으로 틀렸던 게 언제였어? 그때 빈칸 2개에 뭘 넣었는지, 어떤 줄부터 이상하다고 느꼈는지 순서대로 말해줄 수 있어?」**  
✅ **Mom Test 준수** — 과거 한 번의 실패를 시간순으로 복기하게 함

### Q2
**「마방진 풀 때 실수 많이 해? 대각선 빼먹는 게 제일 큰 문제라고 생각해?」**  
❌ **위반** — 일반화 + 유도 질문(답을 미리 끼워 넣음)  
✅ **고친 버전:** 「마지막으로 틀렸을 때, 대각선 말고 행·열만 먼저 맞췄다고 생각한 적 있어? 그때 왜 ‘거의 다 됐다’고 느꼈어?」

---

## 2. 검증/확인 비용

### Q3
**「빈칸 2개 채운 뒤 ‘다 맞는지’ 확인할 때, 행·열·대각선 중 어디부터 합을 냈어? 각각 몇 번 정도 계산했어?」**  
✅ **Mom Test 준수** — 검증 순서·횟수 등 관찰 가능한 행동

### Q4
**「자동으로 합 34 검증해 주는 기능 있으면 쓸 것 같아? 20분 아낄 수 있을 것 같아?」**  
❌ **위반** — 가상의 기능 + 미래 의견 + 솔루션 제안  
✅ **고친 버전:** 「최근에 틀린 뒤 다시 맞출 때, 합 검증에 대략 몇 분 썼어? 어떤 줄부터 다시 세었어?」

---

## 3. 포기·중단 순간

### Q5
**「4×4 마방진 풀다가 ‘오늘은 여기까지’ 하고 멈춘 적 있어? 그때 격자가 어떤 상태였고, 뭐 때문에 그만뒀어?」**  
✅ **Mom Test 준수** — 중단 시점의 구체적 상황·이유

### Q6
**「너무 어려우면 포기하는 편이야, 아니면 끝까지 하는 편이야?」**  
❌ **위반** — 성격·일반론 질문(거짓 긍정 답 유도)  
✅ **고친 버전:** 「마지막으로 중간에 덮어두거나 과제를 미룬 적 있어? 그때 빈칸이 몇 개 남았고, 뭐가 막혔어?」

---

## 4. 같은 문제를 다시 풀 때

### Q7
**「같은 4×4 문제(빈칸 2개)를 두 번째 풀었을 때, 첫 번째랑 뭐가 달랐어? 처음부터 다시 했어, 아니면 숫자 몇 개만 바꿨어?」**  
✅ **Mom Test 준수** — 재시도 시 실제 행동 차이

### Q8
**「두 번째 풀 때 더 빨리 풀 수 있을 것 같아? 첫 번째에서 뭘 배웠을 것 같아?」**  
❌ **위반** — 미래 추측 + “배웠을 것 같다”는 의견  
✅ **고친 버전:** 「같은 문제를 다시 풀었던 마지막 경험 말해줘. 첫 시도 때 넣었던 숫자 중 그대로 둔 게 있었어?」

---

## 5. 다른 사람에게 설명/공유할 때

### Q9
**「친구나 조원에게 이 4×4 문제 풀이를 설명해 준 적 있어? 그때 종이·채팅 중 뭘 썼고, 행·열·대각선 중 뭐부터 보여줬어?」**  
✅ **Mom Test 준수** — 설명 행위의 구체적 매체·순서

### Q10
**「마방진 앱 만들면 친구들한테 공유할 의향 있어? 설명하기 쉬울 것 같아?」**  
❌ **위반** — 아이디어·제품 가정 + 미래 의향  
✅ **고친 버전:** 「마지막으로 누군가에게 ‘여기 0 두 칸 채워줘’라고 맡긴 적 있어? 그 사람이 어디서 막혔다고 했어?」
