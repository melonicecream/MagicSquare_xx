## 2. 인터뷰 답변 (과거 사실)

| 질문 | 답변 요약 |
|------|-----------|

---

## 3. 표면 문제 (금지 — 솔루션 섞인 문장)

아래는 Mom Test에서 **검증하지 않은 가정**으로 분류되는 문장 유형이다.

- “4×4 마방진 **ECB 패턴으로 프로그램** 만든다”
- “`SquareValidator` / `Solver` **클래스를 구현**한다”
- “마방진 **앱·UI**로 빈칸 채우기를 돕는다”
- “TDD로 테스트부터 작성하면 된다” *(인터뷰에서 언급되지 않음 — 금지 질문 유형)*

---

## 4. 진짜 문제 (한 문장)

**4×4 부분 마방진을 손으로 채울 때, 행·열이 맞으면 거의 끝난 것으로 보고 대각선 검증을 늦게 하다가 합이 어긋난 뒤에야 알아차려 약 20분을 다시 맞추는 데 쓴다.**

### 문제 구성 요소

| Mom Test 축 | 관찰 |
|-------------|------|
| **불편** | 대각선 누락을 늦게 발견 |
| **비용** | 약 20분 재작업 |
| **판정** | “대각선 제외하고는 다 맞다”는 잘못된 완료 기준 |
| **재현** | 합을 수동으로 내는 검증 과정에서야 이상 징후 포착 |

---

## 5. Mom Test 증거 3줄

1. “빈칸 2개 넣고 행·열·대각선 합 맞췄는데 **대각선 하나를 빼먹어서 20분 날렸다**”
2. “**남은 빈칸이 거의 없고, 대각선을 제외하곤 다 맞았다고 생각했어**”
3. “**숫자 합을 내서 검증을 하다보니** 알았어” → “**대각선이었어**”
